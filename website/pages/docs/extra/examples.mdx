# Examples

Examples are usually way more helpful than thousand lines of documentation.<br/>
Below you'll find a list of examples showcasing different use cases.

- [**React**](https://github.com/robinweser/elodin/tree/master/examples/react): The React example uses the [CSS in JS](docs/targets/javascript/css-in-js) generator together with [React](https://reactjs.org).
- [**ReasonML**](https://github.com/robinweser/elodin/tree/master/examples/reason):
  The ReasonML example uses the [CSS in Reason](docs/targets/reasonml/css-in-reason) generator together with [ReasonReact](https://reasonml.github.io/reason-react/).

Also, **this website** is built using Elodin and ReasonML! Check the [source code](https://github.com/robinweser/elodin/tree/master/website).
